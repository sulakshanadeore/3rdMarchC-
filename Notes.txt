ExceptionHandling
----------------

Handling unusual or rare behaviour of the application or the user.
In .NET -- structured exception handling.
There is structure(block) for the exception to be handled
ie the program is written in pre defined manner
try
{
}
catch()
{
}
finally
{

}
------------------
System

1) SystemException
